Place publication PDF files in this folder.

Current Publication page links are generated with this pattern:
- yeti-view-points-<itemIndex>-<downloadIndex>.pdf
- yeti-views-<itemIndex>-<downloadIndex>.pdf
- misc-publications-<itemIndex>-<downloadIndex>.pdf

Example:
- assets/publications/yeti-view-points-1-1.pdf
- assets/publications/yeti-view-points-12-3.pdf

If a file is missing, browser download will fail until the PDF is added.
